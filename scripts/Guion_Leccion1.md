## Guion Lección 1: ¿Qué es Java y la Programación? + Variables y Tipos de Datos

Hola y bienvenido. En esta lección vas a aprender qué es la programación, qué es Java y cómo escribir tus primeras líneas de código. No necesitas instalar nada, solo abre tu navegador web y ya puedes empezar. Vamos.

Primero lo primero: ¿qué es programar? Imagina que tienes un robot en casa y quieres que te prepare un sándwich. Ese robot no piensa por sí solo, tú tienes que darle instrucciones exactas, una por una. Paso 1: toma el pan. Paso 2: pon el jamón encima. Paso 3: cierra el sándwich. Si le das las instrucciones en el orden equivocado, el resultado no va a ser lo que querías. Eso es exactamente lo que hace un programa de computadora: una lista de instrucciones que la computadora sigue en orden, una por una, sin saltarse nada.

Ahora, ¿qué es Java? Java es un lenguaje de programación, es decir, es el idioma que usas para darle instrucciones a la computadora. Java es uno de los lenguajes más usados en el mundo desde hace más de 25 años. Las apps de Android, Minecraft y los sistemas bancarios de todo el mundo están escritos en Java. Aprender Java es una habilidad real que usan profesionales en todo el mundo.

Para escribir código en Java no necesitas instalar nada. Vamos a usar un editor online gratuito. Abre una nueva pestaña y escribe esto en la barra de direcciones: onlinegdb.com/online_java_compiler. El único botón que necesitas conocer por ahora es el botón verde que dice Run. Ese botón ejecuta tu código.

Primer programa. Borra todo lo que hay en el editor y escribe el código de Hola Mundo. Presiona Run y mira qué aparece abajo: ¡Hola Mundo!

Ahora te explico qué significa cada parte. public class Main es la caja que contiene tu programa. public static void main es el punto de entrada, donde Java empieza a ejecutar. System.out.println es la instrucción más importante de esta lección, significa mostrar esto en pantalla. Todo lo que pones entre paréntesis y entre comillas aparece en la consola. Las llaves agrupan instrucciones.

Ahora viene algo muy útil: las variables. Una variable es como una cajita con una etiqueta donde guardas información para usarla después. En Java existen 4 tipos básicos: int para números enteros, double para números con decimales, String para texto siempre entre comillas dobles, y boolean que solo puede ser true o false.

Fíjate en estas cosas importantes: String lleva S mayúscula, si la escribes con minúscula Java no la reconoce. El texto de String siempre va entre comillas dobles. Los números int y double no llevan comillas. El símbolo más sirve para unir texto con variables. Las líneas que empiezan con doble slash son comentarios, Java las ignora. Cada instrucción termina con punto y coma, si lo olvidas Java te da error.

Pausa el video aquí. Cambia los valores: pon tu nombre, tu edad real y un promedio inventado. Presiona Run y verifica que funciona. Regresa cuando termines.

Antes de terminar te muestro los errores más frecuentes. Error 1: olvidar el punto y coma al final de cada instrucción. Error 2: escribir String con s minúscula, Java distingue mayúsculas de minúsculas. Error 3: poner un número entre comillas, si haces eso Java lo trata como texto y no como número.

Cuando veas un error en rojo en OnlineGDB, no te asustes. Lee el mensaje de error y revisa estas tres cosas: punto y coma, mayúsculas y comillas.

Llegaste al final de la Lección 1. Aprendiste qué es programar, qué es Java, cómo usar System.out.println, qué son las variables y los 4 tipos básicos: int, double, String y boolean.
